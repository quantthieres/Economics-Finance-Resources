# Economics & Finance Resources

A curated collection of resources on economics, econometrics, finance, energy,
commodities, data and quantitative methods.

The purpose of this repository is to organize useful materials for study,
research and professional development, while making them accessible to
students, researchers, analysts and other interested readers.
