# Repositories

Curated repositories containing code, courses, notebooks, datasets and learning
materials related to economics, finance and quantitative methods.

## Quantitative Finance

### Awesome Quant

- **Author:** Wilson Freitas and contributors
- **Type:** Curated repository
- **Topics:** Quantitative finance, pricing, backtesting, portfolio optimization,
  risk analysis, time series, market data and financial libraries
- **Languages:** Python, R, Julia, Java and others
- **Level:** All levels
- **Description:** A large curated collection of libraries, packages, tools,
  datasets and educational resources for quantitative finance.
- **Why it is useful:** It serves as a broad reference map for discovering
  software, data sources and specialized repositories across several areas of
  quantitative finance.
- **Link:** [Access Awesome Quant](https://github.com/wilsonfreitas/awesome-quant)

### Computational Finance Course

- **Author:** Lech A. Grzelak
- **Type:** Course repository
- **Topics:** Option pricing, stochastic processes, implied volatility,
  jump processes, stochastic volatility, Fourier methods, Monte Carlo,
  hedging and exotic derivatives
- **Language:** Python
- **Level:** Intermediate to advanced
- **Description:** Course materials and Python implementations for a
  computational finance course based on mathematical modeling and numerical
  methods in finance.
- **Why it is useful:** It combines theoretical lectures with practical Python
  implementations of important derivative-pricing models and simulation
  techniques.
- **Link:** [Access the Computational Finance Course](https://github.com/LechGrzelak/Computational-Finance-Course)

### Financial Models and Numerical Methods

- **Author:** Andrea Cantaro
- **Type:** Jupyter notebook collection
- **Topics:** Black–Scholes, stochastic differential equations, Heston model,
  Lévy processes, Fourier methods, partial differential equations, Kalman
  filters, volatility and portfolio optimization
- **Language:** Python
- **Level:** Intermediate to advanced
- **Description:** A collection of interactive notebooks demonstrating
  numerical methods and financial models through practical Python
  implementations.
- **Why it is useful:** It provides well-documented, executable examples of
  methods that are often difficult to find in a practical and accessible
  format.
- **Prerequisites:** Basic financial mathematics, statistics, stochastic
  calculus and Python
- **Link:** [Access Financial Models and Numerical Methods](https://github.com/cantaro86/Financial-Models-Numerical-Methods)

## Machine Learning and Trading

### Machine Learning for Trading

- **Author:** Stefan Jansen
- **Type:** Book companion repository and applied codebase
- **Topics:** Financial data, feature engineering, machine learning,
  backtesting, portfolio construction, transaction costs, risk management,
  causal machine learning, reinforcement learning and live deployment
- **Language:** Python
- **Level:** Intermediate to advanced
- **Description:** Code and case studies accompanying *Machine Learning for
  Trading*, organized around the full workflow from data sourcing and strategy
  research to backtesting, deployment and monitoring.
- **Why it is useful:** It offers an end-to-end framework for developing and
  evaluating machine-learning-driven trading strategies while emphasizing
  overfitting control, validation and operational implementation.
- **Link:** [Access Machine Learning for Trading](https://github.com/stefan-jansen/machine-learning-for-trading)
